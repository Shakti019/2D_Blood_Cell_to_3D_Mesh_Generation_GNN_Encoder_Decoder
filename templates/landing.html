What makes it unique

This pipeline converts flat 2D microscopy images into spatially aware 3D graphs — then runs a graph neural network to classify cell type.

6-Face Cubic Projection

Nucleus pixels are subsampled and mapped onto all 6 faces of a cube using polar coordinates. Delaunay triangulation builds the surface mesh.

GATv2 Graph Neural Network

Three layers of multi-head GATv2Conv with GraphNorm, followed by dual global pooling (mean + max) and a 3-layer GELU MLP classifier.

Interactive Mesh + Table Export

Explore the 3D nucleus mesh interactively (Solid / Nodes / Mesh+Nodes views). Open the full edge data table or download it as CSV.

HSV Nucleus Segmentation

Auto-segments the nucleus via Otsu thresholding on the HSV saturation channel, then isolates the largest contour via morphological cleanup.

Dual-Loss Training

Trained with NLLLoss (classification) + MSELoss (autoencoder reconstruction) for 100 epochs with AdamW + ReduceLROnPlateau scheduler.

FastAPI + Plotly.js

Python FastAPI backend with Jinja2 templates, PyTorch inference on CPU/GPU, and Plotly 2.32 for interactive WebGL 3D rendering.

How it works

Five stages from pixel to prediction

1

Upload microscopy image

Drop or browse a JPG/PNG blood cell image. Supported: basophil, eosinophil, erythroblast, IG, lymphocyte, monocyte, neutrophil, platelet.

2

Nucleus segmentation

HSV saturation channel → Gaussian blur → Otsu threshold → morphological open/close → largest contour mask. Nucleus pixels and their RGB values are extracted.

3

6-Face cubic 3D projection

Subsampled pixels (step=60) are centred and mapped onto 6 cube faces (Front/Back/Right/Left/Top/Bottom) using polar offset. Delaunay triangulation creates the surface mesh for both visualization and graph edges.

4

PyG graph construction

Edge records sampled at idx%10==0 build a PyTorch Geometric Data object. Node features: [Z-norm X, Z-norm Y, Z-norm Z, R/255, G/255, B/255]. Edge attrs: [dist, avg_R, avg_G, avg_B].

5

GNN inference → result

BloodCellHybridGNN encodes the graph, pools to a graph-level signature, and passes it through the MLP classifier to output 8-class softmax probabilities. Results shown with confidence bars and interactive 3D viewer.
